---
title: "Quickstart"
description: "Get up and running with BeeAI in minutes"
---

Follow these simple steps to start using BeeAI:

<Steps>
   <Step title="Install BeeAI">

Install BeeAI using [Homebrew](https://brew.sh/) (recommended for macOS and Linux)

```sh
brew install i-am-bee/beeai/beeai
beeai platform start
```

See [Installation & Setup](/introduction/installation) for alternative methods including Windows instructions.

   </Step>
   <Step title="Configure your LLM provider"> 
   
```sh
beeai env setup
```

   </Step>
   <Step title="Launch the web interface">

```sh
beeai ui
```

   </Step>
   <Step title="Use from the terminal">

```sh
# List commands
beeai --help

# List all available agents
beeai list

# Run the chat agent
beeai run chat
```

   </Step>
</Steps>
